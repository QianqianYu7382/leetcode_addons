# 安装指南

## 图标设置

扩展需要图标文件才能正常工作。请创建以下尺寸的图标：

- `icon16.png` - 16x16 像素
- `icon48.png` - 48x48 像素  
- `icon128.png` - 128x128 像素

### 快速创建图标的方法：

1. **使用在线工具**：访问 https://www.favicon-generator.org/ 或类似工具，上传一个图标即可生成多种尺寸

2. **使用图片编辑软件**：创建一个图标（可以是LeetCode的logo或者一个书本图标），然后导出为上述尺寸

3. **使用临时图标**：可以从网络上找一个合适的图标，确保版权允许使用

### 图标建议：
- 使用LeetCode相关的图标（如书本、代码符号等）
- 颜色建议使用蓝色或紫色，与界面风格一致
- 确保在小尺寸下也能清晰识别

## 完整安装步骤

1. **准备图标文件**
   - 将图标文件放置在项目根目录
   - 确保文件名为 `icon16.png`, `icon48.png`, `icon128.png`

2. **加载扩展**
   - 打开Chrome浏览器
   - 访问 `chrome://extensions/`
   - 开启右上角的"开发者模式"
   - 点击"加载已解压的扩展程序"
   - 选择本项目文件夹（leetcode_addons）

3. **授权权限**
   - 扩展会请求以下权限：
     - 存储权限（保存做题记录）
     - 通知权限（发送复习提醒）
     - 访问LeetCode网站权限
   - 点击"允许"授权所有权限

4. **开始使用**
   - 访问 https://leetcode.cn 或 https://leetcode.com
   - 开始做题，成功提交后会自动记录
   - 点击扩展图标查看记录和复习计划

## 故障排除

### 扩展无法加载
- 确保所有必需文件都在项目中
- 检查 `manifest.json` 格式是否正确
- 查看浏览器控制台是否有错误信息

### 无法检测到提交
- 确保你在LeetCode网站上
- 尝试刷新页面后重新提交
- 检查扩展是否已启用

### 没有收到通知
- 确保已授权通知权限（chrome://extensions/ -> 扩展详情 -> 权限）
- 检查系统通知设置是否允许Chrome发送通知
